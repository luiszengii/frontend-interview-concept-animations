# Animation quality rubric

Score each dimension 0, 1, or 2. Require at least 12/14 and no zero.

## 1. Concept embodiment

- 0: generic layout could represent unrelated topics.
- 1: some topic-specific labels or objects.
- 2: stage geometry and objects directly embody the mechanism.

## 2. Causal continuity

- 0: steps are disconnected slides.
- 1: some state persists.
- 2: persistent objects make every transition and cause legible.

## 3. Visual differentiation

- 0: duplicates another scene's renderer or composition.
- 1: modifies an existing pattern.
- 2: selects a distinct visual grammar appropriate to the concept.

## 4. Explanation depth

- 0: definition or API list only.
- 1: mechanism plus conclusion.
- 2: conclusion, mechanism, implementation detail, boundary, and interview-ready synthesis.

## 5. Edge and failure behavior

- 0: only the happy path.
- 1: mentions an edge case in narration.
- 2: visualizes failure, trade-off, branch, retry, or non-applicable condition.

## 6. Mobile comprehension

- 0: clipped, hover-dependent, or unreadable.
- 1: responsive but cramped.
- 2: touch-friendly, readable, and preserves the key comparison or flow.

## 7. Traceability and sharing

- 0: no question/source link or stable URL.
- 1: partial link coverage.
- 2: question source, question detail URL, and animation URL are all stable and connected.
